# 🐰 PocketPal Mobile: Dijital Evcil Hayvan Serüveni

![PocketPal Banner](https://img.shields.io/badge/Status-Beta-brightgreen) ![Expo](https://img.shields.io/badge/Built%20with-Expo-000000?logo=expo) ![React Native](https://img.shields.io/badge/Framework-React%20Native-61DAFB?logo=react)

**PocketPal Mobile**, nostaljik dijital hayvan (Tamagotchi) deneyimini modern, premium ve oyunlaştırılmış bir mobil uygulama formatında sunar. Sevimli bir arkadaş edinin, onu besleyin, oyunlar oynayın ve birlikte büyüyün!

---

## 🌟 Oyunlaştırma ve Özellikler

PocketPal, kullanıcıyı sürekli etkileşimde tutan zengin bir oyunlaştırma dünyası sunar:

*   **Dinamik Stat Yönetimi**: Tokluk, Neşe ve Temizlik barlarını takip edin. Petinizin ruh hali statlarına göre gerçek zamanlı olarak değişir (Mutlu, Aç, Kirli, Beklemede).
*   **XP ve Seviye Sistemi**: Petinizle ilgilendikçe XP kazanın, seviye atlayın ve yeni özelliklerin kilidini açın.
*   **Gelişmiş Oyun Merkezi (Mini-Games)**: 
    - **Yemek Topla! 🍕**: Hızla düşen yemekleri yakalayarak puan topla.
    - **Balon Patlat! 🎈**: Zaman dolmadan balonları patlatmaya çalış.
    - **Hafıza Oyunu 🧠**: Çiftleri bul ve zihnini tazele.
    - **Yıldız Yakala ⭐** & **Tıklama Yarışı ⚡**.
*   **Gelişmiş Envanter Sistemi**: Mağazadan yemek satın alın ve envanterinizde saklayın. Sadece elinizde olan yemekleri petinize yedirebilirsiniz.
*   **Özelleştirme (Shop & Accessories)**: Kazandığınız coinlerle tac, gözlük veya kurdele gibi aksesuarlar alın veya evinizin duvar kağıdını değiştirin.

## 🛠️ Kurulum ve Çalıştırma (Installation & Run)

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:

### Gereksinimler
- Node.js (LTS sürümü önerilir)
- npm veya yarn
- Expo Go (Mobil cihazda test etmek için Apple App Store veya Google Play Store'dan indirin)

### Adımlar

1.  **Depoyu Klonlayın:**
    ```bash
    git clone https://github.com/elcinx/PocketPal-Mobile.git
    cd PocketPal-Mobile
    ```

2.  **Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    ```

3.  **Uygulamayı Başlatın:**
    ```bash
    npx expo start --tunnel
    ```

4.  **Cihazda Çalıştırın:**
    - Terminalde çıkan **QR kodu** telefonunuzdaki **Expo Go** uygulaması (Android) veya **Kamera** (iOS) ile taratın.
